# Jihan Raihana — Portfolio

The personal portfolio of **Jihan Raihana**, a product designer working end-to-end
with an AI-augmented workflow. Built around a "Fieldwork" theme — cartography meets
field notes.

🔗 **Live:** _(coming soon)_

## Pages

- **Home** — hero, a plotted "territory map", and selected work
- **Synetic** — case study: a healthcare EMR (B2B SaaS)
- **Lilist** — case study: a consumer finance / shopping-list app (with live in-page simulators)
- **About** — the route so far, and how I work

## Built with

- [Astro](https://astro.build) — static site, zero client framework
- Plain CSS with design tokens (warm editorial palette: Playfair Display · Inter · Geist Mono)
- A tiny vanilla-JS motion layer (scroll reveals, a scroll route-line, radar pings,
  breathing contours) — all `prefers-reduced-motion` aware

## Run locally

```sh
npm install
npm run dev      # http://localhost:4321
npm run build    # outputs to ./dist
```
